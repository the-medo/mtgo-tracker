

## Getting Started

.env file:
```
GITHUB_ID=...
GITHUB_SECRET=...
GOOGLE_CLIENT_ID=...
GOOGLE_CLIENT_SECRET=...
NEXTAUTH_SECRET=...
DATABASE_URL="postgresql://fetchbookdb_connection_string/fetchbookdb?sslmode=require"
NEXT_PUBLIC_VERSION=local
NEXT_PUBLIC_API_URL="http://localhost:3000"
```


Run:
```bash
npm install
npm run dev
```
